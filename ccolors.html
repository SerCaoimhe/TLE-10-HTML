Colors in CSS

Colors can be applied with CSS in order to stylize HTML. There are a variety of ways to apply colors to different parts e.g. border, background, element content, box-shadow, outline of an element, each of which has its own property for color.

Color Properties for Different Parts
Part Property
Background background-color
Border border-color
Text color

Color properties can be combined with any of the color value types discussed below.

Example: color properties and value types

Syntax: 
<div style="
  background-color: lightcoral;
  border-color: rgb(0,0,0);
  color: #ffffff">
  1 2 3 4 5 6 7 8 9 0
</div>
Preview:

Color Values

Types

Color values can be specified in a variety of ways. These color values can be applied freely. More than one way can be used in an HTML document.

These are the different value types that can be used in CSS:
