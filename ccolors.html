Colors in CSS

Colors can be applied with CSS in order to stylize HTML. There are a variety of ways to apply colors to elements, each of which has its own property for color.

Color Properties for Different Elements
Element Property
Background background-color
Border border-color
Text color

Color properties can be combined with any of the color value types discussed below.

Example: color properties and value types 

background-color: white
border-color: rgb(0,0,0)
color: #000000
