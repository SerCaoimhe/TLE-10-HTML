CSS Examples

In this HTML document, all three types of CSS are applied: 1) external, 2) internal, and 3) inline.

Pay attention to which among the three is implemented when there is an overlapping rule. Specifically, a rule for the body element is being applied externally, internally, and inline at the same time. Specificity will be considered by the browser when a declaration is conflicting among the CSS types. In this case, only the background-color property is conflicting. color and font-family are implemented as is. 

<head>
  <title>CSS Examples</title>
  <link rel="stylesheet" href="sample-external.css"> 
  <style>
    body {
      background-color: white;
    }
  </style>
</head>
<body style="background-color: rgb(254, 227, 91);">
...
</body>
