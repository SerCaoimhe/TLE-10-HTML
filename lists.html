What is a list?

A list enumerates items within a group.

Lists can be combined and nested.

Types of Lists

There are three types of lists, each with their appropriate use.

Unordered List

The unordered list element is used to enumerate items in no particular order using bullets as the default list marker. It is begun with the unordered list <ul> tag and filled with the list item <li> tag.

The list marker may be changed from a bullet to another type using the style attribute with the property "list-style-type" and the syntax: <li style="list-style-type:x;"> where x is any of the following values: disc, square, circle, or none. The "disc" property value will return the default bullet marker.

Syntax:

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
    <li style="list-style-type: disc;">Item 4 (disc marker)</li>
    <li style="list-style-type: square;">Item 5 (square marker)</li>
    <li style="list-style-type: circle;">Item 6 (circle marker)</li>
    <li style="list-style-type: none;">Item 7 (no marker)</li>
</ul>

Preview:

  • Item 2
  • Item 3
  • Item 4 (disc marker)
  • Item 5 (square marker)
  • Item 6 (circle marker)
  • Item 7 (no marker)

Ordered List

The ordered list element is used to enumerate items in a particular order using a defined numbering style. It is begun with the ordered list <ol> tag and filled with the list item <li> tag.

The attribute "type" defines the numbering system (1,I,i,A,a), while "start" defines the starting number. If no attributes are used, the list will use the default (1 starting from 1).

Syntax:

<ol type="1" start="1">
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ol>

Preview:

  1. Item 1
  2. Item 2
  3. Item 3

Description List

The description list element is used to enumerate and describe items. It is begun with the description list <dl> tag and filled with the definition term <dt> and description details <dd> tags.

The description details typically begin with a list marker such as a bullet or hyphen, but this is not required. Just remember that the list will display whatever the element content is (including the list marker, if any).

When writing in code, separate each definition term with a blank line for clarity.

Syntax:

<dl>
    <dt>Item 1</dt>
    <dd>Item 1 description 1 (no list marker)</dd>
    <dd>- Item 1 description 2 (hyphen list marker)</dd>
