What is a list?

A list enumerates items belonging to a group.

Lists can be combined with other types of list and nested. comprised of sublists

Types of Lists

There are three types of lists, each with its appropriate use.

1. Unordered List

The unordered list element is used to enumerate items in no particular order using bullets as the default list marker symbol that marks each item . It is begun with the unordered list <ul> tag and filled with the list item <li> tag.

The list marker may be changed from a bullet to another type using the style attribute see Structure>Attribute with the list-style-type property and corresponding property value with the syntax: <ul style="list-style-type:x;"> where x is any of the following values: disc, default, shaded circle aka "bullet" circle, unshaded circle square, shaded square none no list marker . The disc value will return the default bullet marker.

Attribute Property Value
<ul style= "list-style-type: x;" >

Possible list-style-types:

  • default (no list-style-type value specified)
  • disc
  • circle
  • square
  • none

The list-style-type property can be defined for either the entire list by placing it in the <ul> tag or for a specific item by placing it in the <li> tag.

All items in the list will stick to the list-style-type value defined for the list unless overridden when a general rule is replaced by a more specific rule by a custom list-style-type value in an item (Example 2).

Example 1: property defined for entire list

Syntax: 
<ul style="list-style-type: circle;">
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>

Example 2: property defined for specific items

Syntax: 
<ul><!--list-tyle-type is not defined, so list follows default bullet until overridden-->
  <li>Item 1</li>
  <li>Item 2</li>
  <li style="list-style-type: disc;">Item 3</li>
  <li style="list-style-type: circle;">Item 4</li>

  <li style="list-style-type: none;">Item 6</li>

2. Ordered List

The ordered list element is used to enumerate items in a particular order using Arabic numerals e.g. -1, 0, 1 as the default numbering style. It is begun with the ordered list <ol> tag and filled with the list item <li> tag.

The type attribute defines the numbering system while start defines the starting value with the syntax: <ul type="x" start="y"> where x is any of the following values: 1 for Arabic numerals , I for uppercase Roman numerals , i for lowercase Roman numerals , A for uppercase alphabet , a for lowercase alphabet ; and y is any natural number (integer in the case of Arabic numerals).

If no attributes are used, the list will use the default Arabic numerals starting from 1.

<ol type= "x" >

The type attribute can be defined for either the entire list by placing it in the <ol> tag or for a specific item by placing it in the <li> tag.

All items in the list will stick to the type value defined for the list unless overridden by a custom type value in an item (Example 2).

Example 1: type defined for entire list

Syntax: 
<ol type="A" start="1">
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ol>

3. Description List

The description list element is used to enumerate and describe items. It is begun with the description list <dl> tag and filled with the definition term <dt> and description details <dd> tag.

The description details typically begin with a list marker such as a bullet or hyphen, but this is not required. Just remember that the list will display whatever the element content is (including the list marker, if any).

When writing in code, separate each definition term with a blank line for clarity.

Syntax:

<dl>
  <dt>Item 1</dt>
  <dd>Item 1 description 1 (no list marker)</dd>
  <dd>- Item 1 description 2 (hyphen list marker)</dd>
  
  <dt>Item 2</dt>

Nested Lists

A list can be comprised of smaller lists inside. Simply begin another list under an item in the current parent list. Different types of lists may comprise a nested list.

ü§ì Author's Markup: Always remember the "CEO rule": first one in, last one out. Example: <ol><li><ul><li>...</li></ul></li></ol>

Example 1: nested list using only one type (unordered list)

Syntax: 
<!--spaces between items added for better legibility but not required-->
<h3>Why Sir Derik is the best:</h3>
<!--START of main UNORDERED list-->
<ul>
  
  <li>He is tall<!--item is not closed until the sublist is closed-->
  <!--Notice how another list is opened under an item in the original list-->    
  <!--START of sublist-->    
    <ul style="list-style-type: circle;">
      <li>Can turn on the airconditioner without stepping on a chair, which is unsanitary</li>
      <li>Can turn off the airconditioner without stepping on a chair, which is still unsanitary</li>
      <li>Tallest Berean thus far</li>
    <!--END of sublist-->
    </ul>
  </li><!--item is now closed because the sublist is closed-->
  
  <li>He is handsome<!--item is not closed until the sublist is closed-->
    <!--START of sublist-->
    <ul style="list-style-type: circle;">
      <li>Confirmed by mother, grandmother, and aunts</li>
      <li>Referred to as "pogi" by vendors and conductors</li>
      <li>Feels people in the LRT stealing glances</li>
      <li>Has a strong gut feeling about it</li>
    <!--END of sublist-->
    </ul>
  </li><!--item is now closed because the sublist is closed-->
  
  <li>He is smart<!--item is not closed until the sublist is closed-->
    <!--START of sublist-->
    <ul style="list-style-type: circle;">
      <li>Knows the solution and answer to 1+1</li>
      <li>Has memorized the multiplication table up to 16x16</li>
    <!--END of sublist-->
    </ul>
  </li><!--item is now closed because the sublist is closed-->
  
  <li>He is humble<!--item is not closed until the sublist is closed-->
    <!--START of sublist-->
    <ul style="list-style-type: circle;">
      <li>The humblest in the world</li>
      <li>Does not brag about his many great qualities</li>
    <!--END of sublist-->
    </ul>
  </li><!--item is now closed because the sublist is closed-->
  
  <li>He is <s>Superman</s> helpful</li>
  <li>He <s>loves</s> likes his advisory class</li>

<!--END of main UNORDERED list-->
</ul>
