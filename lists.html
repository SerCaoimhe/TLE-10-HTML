What is a list?

A list enumerates items belonging to a group.

Lists can be combined with other types of list and nested. placed inside another list

Types of Lists

There are three types of lists, each with its appropriate use.

1. Unordered List

The unordered list element is used to enumerate items in no particular order using bullets as the default list marker symbol that marks each item . It is begun with the unordered list <ul> tag and filled with the list item <li> tag.

The list marker may be changed from a bullet to another type using the style attribute a CSS attribute with the property one of many definable variables of the style attribtue list-style-type and corresponding property value with the syntax: <ul style="list-style-type:x;"> where x is any of the following values: disc default, shaded circle aka "bullet" , circle unshaded circle , square shaded circle , none no list marker . The "disc" value will return the default bullet marker.

Attribute Property Value
<ul style= "list-style-type: x;" >

The list-style-type can be defined for either the entire list by placing it in the <ul> tag or for a specific item by placing it in the <li> tag.

Example 1: property defined for entire list

Syntax: 
<ul style="list-style-type: circle;">
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
  <li>Item 4</li>
  <li>Item 5</li>
</ul>

Example 2: property defined for specific items

Syntax: 
<ul>
  <li>Item 1</li>
  <li style="list-style-type: disc;">Item 2 (disc marker)</li>
  <li style="list-style-type: square;">Item 3 (square marker)</li>
  <li style="list-style-type: circle;">Item 4 (circle marker)</li>
  <li style="list-style-type: none;">Item 5 (no marker)</li>
</ul>

2. Ordered List

The ordered list element is used to enumerate items in a particular order using a defined numbering style. It is begun with the ordered list <ol> tag and filled with the list item <li> tag.

The attribute "type" defines the numbering system (1,I,i,A,a), while "start" defines the starting number. If no attributes are used, the list will use the default (1 starting from 1).

Syntax:

<ol type="1" start="1">
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ol>

Preview:

  1. Item 1
  2. Item 2
  3. Item 3

3. Description List

The description list element is used to enumerate and describe items. It is begun with the description list <dl> tag and filled with the definition term <dt> and description details <dd> tags.

The description details typically begin with a list marker such as a bullet or hyphen, but this is not required. Just remember that the list will display whatever the element content is (including the list marker, if any).

When writing in code, separate each definition term with a blank line for clarity.

Syntax:

<dl>
  <dt>Item 1</dt>
  <dd>Item 1 description 1 (no list marker)</dd>
  <dd>- Item 1 description 2 (hyphen list marker)</dd>

Nested Lists

A list can be comprised of smaller lists inside. Simply begin another list under an item in the current parent list. Different types of lists may comprise a nested list.

Example 1: nested list using only one type (unordered list)

Syntax: 
<!--spaces between items added for better legibility, but not required-->
<h3>Why Sir Derik is the best:</h3>
<!--START of main UNORDERED list-->
<ul>
  
  <li>He is tall<!--item is not closed until the sublist is closed-->
  <!--Notice how another list is opened under an item in the original list-->    
  <!--START of sublist-->    
    <ul style="list-style-type: circle;">
      <li>Can turn on the airconditioner without stepping on a chair, which is unsanitary</li>
      <li>Can turn off the airconditioner without stepping on a chair, which is still unsanitary</li>
      <li>Tallest Berean thus far</li>
    <!--END of sublist-->
    </ul>
  </li><!--item is now closed because the sublist is closed-->
  
  <li>He is handsome<!--item is not closed until the sublist is closed-->
    <!--START of sublist-->
    <ul style="list-style-type: circle;">
      <li>Confirmed by mother, grandmother, and aunts</li>
      <li>Referred to as "pogi" by vendors and conductors</li>
      <li>Feels people in the LRT stealing glances</li>
      <li>Has a strong gut feeling about it</li>
    <!--END of sublist-->
    </ul>
  </li><!--item is now closed because the sublist is closed-->
  
  <li>He is smart<!--item is not closed until the sublist is closed-->
    <!--START of sublist-->
    <ul style="list-style-type: circle;">
      <li>Knows the solution and answer to 1+1</li>
      <li>Has memorized the multiplication table up to 16x16</li>
    <!--END of sublist-->
    </ul>
  </li><!--item is now closed because the sublist is closed-->
  
  <li>He is humble<!--item is not closed until the sublist is closed-->
    <!--START of sublist-->
    <ul style="list-style-type: circle;">
      <li>The humblest in the world</li>
      <li>Does not brag about his many great qualities</li>
    <!--END of sublist-->
    </ul>
  </li><!--item is now closed because the sublist is closed-->
  
  <li>He is <s>Superman</s> helpful</li>
  <li>He <s>loves</s> likes his advisory class</li>

<!--END of main UNORDERED list-->
</ul>
