Introduction to Programming

What is Programming

Programming is a broad field that is fundamentally involved in computing the development and operation of computers . It encompasses activities such as planning, designing, and implementing computer solutions. While often used interchangeably with "coding", the two concepts have distinct differences.

Coding is an aspect of programming that involves the encoding of a set of instructions that is understood and executable by a computer. Coding includes various languages: programming e.g. Python, JavaScript, Java, C++ , markup e.g. HTML, XML , and style sheet e.g. CSS, XSL .

What is HTML

HTML stands for hypertext beyond text markup adds meaning and context language has vocabulary, rules, and syntax . It is one of many languages used in coding and is specifically used for web development. While not a programming language, it is still a coding language because it provides a set of instructions on how browsers should display content.

It is one of the three primary web development languages which include CSS cascading stylsheet and JS JavaScript . HTML defines the structure and content, CSS defines the presentation and layout, while JS provides the functionality and interactivity.
