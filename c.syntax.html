Syntax

If the basic building block of html is the element, in CSS, it is the rule. The rule itself can be split into two main segments, the selector and the declaration block, which in turn is composed of at least one declaration.

A declaration is composed of the property and value pair. Multiple property-value pairs are separated by semicolons.

Parts of the CSS rule:
  1. Selector x
  2. Declaration Block
    1. Open curly bracket {
    2. Declaration
      1. Property y
      2. Colon :
      3. Value z
      4. Semicolon ;
    3. Closing curly bracket }
Selector Declaration Block
x {y1: z1; y2: z2;}

The syntax in the table above applies when the rule is put in a single line. However, for more detailed rules, it is recommended to split it into several lines for clarity.

Multiline rule syntax: 

        x {
        y1: z1;
        y2: z2;
        y3: z3a z3b;
        }

Properties can have multiple values as shown in the third property, y3.

Selectors

The selector is an essential part of CSS. It is the part of the rule that determines which element or elements are affected. There are several types of selectors, making styling highly flexible and precise.

Individual

Selector Syntax Purpose
Universal * {declaration block} selects all elements
Element element {declaration block} selects the defined element
Class .class {declaration block} selects elements with the defined class attribute value
Element-Class element.class {declaration block} selects only the defined element with the defined class attribute value
ID #id {declaration block} selects the elements with the defined id attribute value

Element Selector

Syntax: 

 h3  {
   font-style: italic;
 }
 p  {
   text-transform: uppercase;
 }
 table  {
   background-color: aquamarine;
 }

Class Selector

Syntax: 

.tnr {
  font-family:'Times New Roman';
  background-color: aquamarine;
}
.courier {
  font-family: 'Courier New';
  background-color: rgb(247, 150, 203);
}
Preview:

Element-Class Selector

Syntax: 

h3.blue {
background-color: blue;
color: white;
}
td.tnr {
font-family: 'Times New Roman';
background-color: salmon;
color: white;
font-weight: bold;
}

ID Selector

Syntax: 

#yellow {
  background-color: black;
  color: yellow;
}
#salmon {
  background-color: black;
  color: salmon;
}

Group

Multiple selectors may be used in one rule to apply the same style to all of them. This makes the code less cluttered and more efficient. To use group selectors, simply separate the selectors with commas.

Example 1: multiple element selectors

Syntax: 

h3, h4 {
  background-color: black;
  color: yellow;
}
p, table {
  background-color: rgb(114, 250, 114);
}
Preview:
Example 2: mixed selectors

Syntax: 

.und, #und1 {
  text-decoration: underline;
}
h4, .strike {
  text-decoration: line-through;
  background-color:rgb(254, 227, 91);
}
