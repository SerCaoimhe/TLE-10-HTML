What is an Editor

An editor is a program that enables one to write code in several languages including, but not limited to: HTML, CSS, JS, Python, C++.

Visual Studio Code

Visual Studio Code is a program developed by Microsoft and is the most commonly used editor in the software development field.

Inputs

When coding, one should be able to maximize the use of the keyboard for efficiency. virtually all actions and features in Visual Studio Code can be accessed using keyboard inputs. aka "shortcuts"; usually consist of two or more keys pressed together or in sequence

The mouse remains an indispensible tool with its three standard buttons( left mouse button, LMB scroll wheel, and right mouse button) RMB which can be used also to apply actions and access features.

Most keyboard and mouse inputs work independent of each other, but some require their combined use. These are some useful keyboard and mouse inputs:
