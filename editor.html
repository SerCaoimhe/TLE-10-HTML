What is an Editor

An editor is a program that enables one to write code in several languages including, but not limited to: HTML, CSS, JS, Python, C++.

Visual Studio Code

Visual Studio Code is a program developed by Microsoft and is the most commonly used editor in the software development field.

Inputs

These are some useful keyboard and mouse inputs:

File Management

Action Keyboard Input Mouse Input
Create New File Type Ctrl + Alt + Win + N
Create New file Ctrl + N
Open File Ctrl + O
Open Folder Ctrl + K, Ctrl + O
Save Ctrl + S
Save As Ctrl + Shift + S
Save All Ctrl + K, Ctrl + S
Go to File Ctrl + P

Navigation

Action Keyboard Input Mouse Input
Zoom In/Out Ctrl +/Ctrl + -
Reset Zoom Ctrl + 0 (numpad)
Scroll Up/Down Ctrl + ↑/Ctrl + ↓ Scroll Wheel Up/Scroll Wheel Down
Next/Previous Tab Ctrl + Tab/Ctrl + Shift + Tab
Close Tab Ctrl + W/F4
Close All Tabs Ctrl + K, Ctrl + W
Find Ctrl + F
Find Next/Previous F3/Shift + F3
Find in Selection Alt + L
Move one Character at a time ←, →
Move one Word at a time Ctrl + ←, Ctrl + →
Move one Line at a time ↑, ↓
Add Cursor Above/Below Ctrl + Alt + ↑/Ctrl + Alt + ↓
Swap Line Above/Below Alt + ↑/Alt + ↓

Selection and Editing

Action Keyboard Input Mouse Input
Select All Ctrl + A
Select one Character at a time Shift + ←,Shift + →, LMB + drag
Select one Word at a time Ctrl + Shift + ←, Ctrl + Shift + →, Double LMB
Select one Line at a time Shift + ↓,Shift + ↑, Triple LMB
Delete Previous/Next Character Backspace/Delete
Undo/Redo Ctrl + Z/Ctrl + Y
Insert Indentation Tab 4 spaces by default; 2 spaces preferred (can be changed in settings)
Copy/Cut Ctrl + C/Ctrl + X
Paste Ctrl + V
