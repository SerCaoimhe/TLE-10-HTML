Text Elements

1. Heading

The heading element is used for headings and subheadings. It is applied using the tags <h1> to <h6> which define hierarchical levels for text, from highest (<h1>) to lowest (<h6>).

Syntax:

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>

2. Paragraph

The paragraph element is used for bodies of text. It is applied using the tag <p> which automatically creates a new line.

Syntax:

<p>This is one paragraph.</p>
<p>This is another paragraph.</p>

Breaks

1. Line Break

The line break element forces the content to continue on a new line within the paragraph. It is applied using the empty tag <br>. An empty tag is also known as a self-closing tag, or preferably void tag. This does not have an element content nor an end tag.

Syntax:

<p>This is one paragraph but you want a break after this.<br>
   This still belongs to the same paragraph.</p>
   <p>This is another paragraph.</p>

Preview:

This is one paragraph but you want a break after this.
This still belongs to the same paragraph.

This is another paragraph.

2. Horizontal Rule

The horizontal rule element indicates a semantic break in the content and is indicated by a horizontal line between different sections. It is applied using the void tag <hr>.

Syntax:

<h2>This is a main heading.</h2>
  <p>This is a paragraph.</p>
<hr>
<h2>This is different main heading.</h2>
  <p>This is a paragraph.</p>

Preview:

This is a main heading.

This is a paragraph.

This is a different main heading.

This is a paragraph.

Preformatted Text

The preformatted text element defines text to be displayed with a fixed-width or monospace font (all letters and spaces have the same width) and with the spaces and line breaks preserved.

Syntax:

<p>This is regular 
                    
                    text where extra spaces and line breaks are ignored.
</p>
<pre>This is preformatted text where the 
            font is monospace by default and

extra spaces and line breaks will be 

     preserved.
</pre>

Preview:

     preserved.

ASCII Art:

This element is also useful for displaying American Standard Code for Information Interchange (ASCII) art.
