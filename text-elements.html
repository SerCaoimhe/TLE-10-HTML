Text Elements

1. Heading

The heading element is used for headings and subheadings. denote topics in the page It is applied using the heading 1 <h1> to heading 6 <h6> tags which define hierarchical levels for text, from highest (<h1>) to lowest (<h6>).

Syntax:

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>

2. Paragraph

The paragraph element is used for bodies of text. It is applied using the paragraph <p> tag which automatically creates a new line with space above and below.

Syntax:

<p>This is one paragraph.</p><p>This new paragraph is on the same line in the code, but is delegated to the next line in the browser.</p>
<p>This is another paragraph.</p>

Breaks

1. Line Break

The line break element forces the content to continue on a new line within the paragraph. It is applied using the break <br> void tag. does not have an element content nor an end tag A void tag is also known as a self-closing or empty tag.

Syntax:

<p>This is one paragraph but you want a break after this.<br>
  This still belongs to the same paragraph but is forced onto the next line.
</p>
<p>This is another paragraph.</p>

Preview:

This is one paragraph but you want a break after this.
This still belongs to the same paragraph but is forced onto the next line.

This is another paragraph.

2. Horizontal Rule

The horizontal rule element indicates a semantic has meaning and non-visual implication break in the content and is indicated by a horizontal line between different parts of the page. It is applied using the horizontal rule <hr> void tag.

Syntax:

<h2>This is a main heading.</h2>
<p>This is a paragraph.</p>
<hr>
<h2>This is different main heading.</h2>
<p>This is a paragraph.</p>

Preview:

This is a paragraph.

This is a paragraph.

Preformatted Text

The preformatted text element defines text to be displayed with a fixed-width or monospace all characters have the same width font and with the spaces and line breaks in the code preserved. It is applied using the preformatted <pre> tag.

This is useful for presenting code as it is written and not as it is rendered, displayed by the browser as seen in the Syntax sections in this site.

Syntax:

<p>This is regular 
              
              text where extra spaces and line breaks in the code are ignored.
</p>
<pre>This is preformatted text where the 
            font is monospace by default and

extra spaces and line breaks in the code are 

     preserved.
</pre>
