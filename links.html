What is a link?

A link is a concept in computing that refers to a connection between two objects (webpages, files, etc.).

Meanwhile, a hyperlink is a specific type of link created and operated using HTML. In simple terms, a hyperlink is an interactive link beyond than just clickable (e.g.. provides visual feedback, navigable using Tab)

Hyperlink

The hyperlink element inserts a hyperlink inside the HTML document. It is typically in the form of a colored, underlined text but can also be in other elements (e.g. image, button). Another indicator that something is a hyperlink is the pointer's behavior when hovering over it. The pointer aka "cursor"; the arrow you're using now to read this tooltip text typically takes the form of a hand when in front of a hyperlink. Try here.

Hyperlinks by default also display a different color for each condition:

  • blue - unvisited
  • red - activated the instance it is clicked
  • purple - visited

ðŸ¤“ Author's Markup: You can cancel activting a link mid-click by dragging the pointer away before letting go of the left mouse button.

A hyperlink is applied using the anchor <a> tag, the required hyperlink reference href attribute, and the link text interactive text representing the hyperlink element content with the syntax <a href="x">y</a> where x is the destination URL web address (Uniform Resource Locator) or file destination. within local machine and y is the link text.

Attribute Value Element Content
<a href= "x" > y </a>

Another attribute ideally used is target. By default, the linked document will be opened in the same frame, usually means the same browser tab replacing the one being displayed before the link was clicked. The target attribute allows one to set where the linked document will be opened. It can have any one of the following values:

  • _self - default; opens the document in the same frame
  • _blank - opens the document in a new tab
  • _parent - opens the document in the parent frame
  • _top - opens the document in the outermost frame essentially the entire tab

Example 1: target defined as _self

Syntax: 

<p>
  Clicking the link will replace this page with the destination <strong>(simply go back to return)</strong>:
  <a href="https://www.youtube.com/shorts/DgnnbwXQ7Nc" target="_self" title="open the link in the current tab">
    Fried bologna sandwich tutorial
  </a>
</p>
