What is a Link?

A link is a concept in computing that refers to a connection between two objects (webpages, files, etc.).

Meanwhile, a hyperlink is specific type of link created and operated using HTML. In simple terms, a hyperlink is an interactive link beyond than just clickable (e.g.. provides visual feedback, navigable using Tab)

Hyperlink

The hyperlink element inserts a hyperlink inside the HTML document. It is applied using the anchor tag <a>, the required hyperlink reference href attribute, and the link text interactive text representing the hyperlink element content with the syntax <a href="x">y</a> where x is the destination URL web address (Uniform Resource Locator) or file destination. within local machine and y is the link text.

Attribute Value Element Content

   href=
"x" > y </a>

Another attribute ideally used is target. By default, the linked document will be opened in the same frame, usually means the same browser tab replacing the one being displayed before the link was clicked. The target attribute allows one to set where the linked document will be opened. It can have any one of the following values:

  • _self - default; opens document in the same frame
  • _blank - opens the document in a new tab
  • _parent - opens the document in the parent frame
  • _top - opens the document in the outermost frame essentially the entire tab

Example 1: target defined as _self

Syntax

<p>
  Clicking the link will replace this page with the destination (simply go back to return here):
  <a href="https://youtube.com/shorts/DArL8-mqZqs?si=-GFqCSnn4T3E3ky4" target="_self" title="open the link in the current tab">
    Watermelon Sugar PUG!
  </a>
</p>
