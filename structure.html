Syntax

Element

The element is the building block of HTML. It defines the content to be presented by the browser. program or application that displays webpages It is comprised of characters letters, numbers, symbols, and spaces arranged according to a particular syntax. rules governing how a language is structured

Examples of characters:

  • a is a character
  • 1 is a character
  • ! is a character
  • (space) is a character

An element has specific parts that must be present to be understood and executed by the browser. It is initiated by the start tag, aka "open" tag contains element content, the actual content to be presented by the browser and ends with the end tag aka "close" tag .

Parts of an Element

  1. Start Tag
    1. Opening angle bracketaka the "less than symbol" <
    2. Tag nameelement identifier x
    3. Closing angle bracketaka the "greater than symbol" >
  2. Element Content abc
  3. End Tag
    1. Opening angle bracket <
    2. Forward slashindicates closing tag /
    3. Tag name x
    4. Closing angle bracket >
Start Tag Element Content End Tag
<x> abc </x>

Example 1: Paragraph Element

Syntax: 
<p>This is the element content.</p>

Attribute

An attribute is an optional but useful part of an element. It provides additional information that serve to further specify instructions for the browser. They can be applied to all elements. To be applied, the attribute name name of the attribute to be defined and attribute value how the browser presents the element content based on chosen attribute must be encoded inside the start tag.

Parts of an Attribute

  1. Attribute Name typethe text before the equal sign
  2. Attribute Value valuethe text inside the quotes

    3. CSS attributes can be used in HTML, and they have an additional component:

  3. Propertyadditional component to CSS attributes propertythe text after the equal sign
  4. Property Value valuethe text after the colon
Tag Name Attribute Name Attribute Value/Property Property Value
HTML <ol type= "value"
CSS <ul style= "property: value"

Example 1: HTML attributes for an ordered list

The attributes applied are "type" and "start". The defined values for each are inside the quotation marks.

Syntax: 
<ol typeattribute="Ivalue" start="1">
        <li>Item 1</li>
        <li>Item 2</li>     
    </ol>

Example 2: CSS attribute and properties for an unordered list

The attribute applied is "style", while the properties are "line-style-type" and "background-color". The defined value for each property is placed after the colon.

Multiple properties can be included inside one CSS attribute and are separated by semicolons.

Syntax: 
<ul styleattribute="list-style-typeproperty: circleproperty value; background-color:#b5d6f3;">
        <li>Item 1</li>
        <li>Item 2</li>     
    </ul>
