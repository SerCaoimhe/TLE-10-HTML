Structure

Parts

Basic HTML code contains the following structural elements:

Element Tags Purpose
Document type declaration <!DOCTYPE html> Indicates to the browser that the document type is HTML
HTML <html></html> Indicates the start and end of the HTML document
Head <head></head> Contains metadata "data about data"; document properties about the HTML document
Title <title></title> Defines the title of the webpage on the browser tab and search results
Body <body></body> Contains all visible elements of the webpage
Syntax 
<!DOCTYPE html>
<html>
<head>
  <title>HTML Basics</title>
</head>

Syntax

Element

The element is the building block of HTML. It defines the content to be presented by the browser. program or application that displays webpages It is comprised of characters letters, numbers, symbols, and spaces arranged according to a particular syntax. rules governing how a language is structured

Examples of characters:

  • a is a character
  • 1 is a character
  • ! is a character
  • (space) is a character

Parts of an Element

An element has specific parts that must be present to be understood and executed by the browser. It is initiated by the start tag, aka "open" tag contains element content, the actual content to be presented by the browser and ends with the end tag aka "close" tag .

  1. Start Tag
    1. Opening angle bracketaka the "less than symbol" <
    2. Tag nameelement identifier x
    3. Closing angle bracketaka the "greater than symbol" >
  2. Element Content abc
  3. End Tag
    1. Opening angle bracket <
    2. Forward slashindicates closing tag /
    3. Tag name x
    4. Closing angle bracket >

Some elements are empty which means that they do not have element content nor end tags (Example 2).

Start Tag Element Content End Tag
Regular Element <x> abc </x>
Empty Element <x>

Attribute

An attribute is an aspect of an element. It provides additional information that serves to further specify instructions for the browser. They can be applied to all elements, but most do not require them, save for some. e.g. <a>, <img>, <iframe>, <input> To be applied, the attribute name name of the attribute to be defined and attribute value how the browser presents the element content based on chosen attribute must be encoded inside the start tag.

Parts of an Attribute

  3. Property propertythe text after the equal sign

Example 1: HTML attributes for an ordered list

The attributes applied are type and start. The defined values for each are inside the quotation marks.

Syntax: 
<ol typeattribute="Ivalue" start="1">
  <li>Item 1</li>
  <li>Item 2</li>     
</ol>

Example 2: Style and properties for an unordered list

The attribute applied is style, while the properties are line-style-type and background-color. The defined value for each property is placed after the colon.

Multiple properties can be included inside the style attribute and are separated by semicolons. Take note that all property:value pairs, separated by semicolons, are enclosed within the one quote pair after the equal sign.

Syntax: 
<ul styleattribute="list-style-typeproperty1: circlevalue1; background-colorproperty2:#b5d6f3value2;">
  <li>Item 1</li>
  <li>Item 2</li>     
</ul>
