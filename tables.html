What is a table?

A table is an object that organizes data into cells using rows (horizontal component) and columns (vertical component) typically within a header and body structure.

Making a Table

  1. Begin a table using the <table> tag.
  2. Use the style attribute and text-align property to set the alignment of the cell content (center, right, left, or justify).
  3. Use the width, cellspacing, and border attributes to set appearance.
  4. To build it, keep adding rows using the table row <tr> tag.
  5. Add columns using either the
    • table header <th> tag if it's for the header
    • table data <td> tag if it's for the body

Syntax:


<table style="text-align:center" width="100%" border="1" cellspacing="0">
    <tr><!--START of 1st row-->
        <th>Header 1</th>
        <th>Header 2</th>
        <th>Header 3</th>
    </tr><!--END of 1st row-->
    <tr><!--START of 2nd row-->
        <td>Detail 1</td>
        <td>Detail 1</td>
        <td>Detail 1</td>                        
    </tr><!--END of 2nd row-->
    <tr><!--START of 3rd row-->
        <td>Detail 2</td>
        <td>Detail 2</td>
        <td>Detail 2</td>                        
    </tr><!--END of 3rd row-->
</table>
