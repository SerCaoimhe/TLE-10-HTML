What is a background?

The background refers to the back-most part of an element. Each element has a background. A paragraph element, for example, has a background independent from the body's. By default, the background is transparent until styled (e.g. with a color).

This paragraph's background is currently transparent, and, in effect, the body's gradient background color is visible. If the background is styled such as in this span element within the paragraph, it will appear accordingly.

Example 1: color background

Syntax: 
<div style=
  "background-color: steelblue;
  width: 250px; 
  height: 289x;
  color:white;
  padding: 10px"
>
  Video on top of steel blue background
  <video 
    src="Multimedia/1000.mp4"
    controls
    autoplay
    width="100%"
    title="1000%">
    1000% dance
  </video>
</div>
Preview:
Video on top of steel blue background

Example 2: image background

Syntax: 
<div style=
  "background-image: url(Multimedia/snoopy.png);
  flex-grow: 1; 
  height: 289px; 
  width: 250px;
  background-repeat: no-repeat;
  padding: 10px"
>
  <span style="background-color: white;">
    Picture on top of PNG background
  </span>
  <br>
  <img src="Multimedia/woodstock.webp" 
    height="40%" 
    style="mix-blend-mode:darken;"
    alt="Woodchuck"
    title="Woodchuck">
</div>

Properties

There are several properties that can be defined in order to style the backgorund.

Background Properties
Property Syntax Description
Background Color background-color: defines the background color
Background Image background-image: defines the background image
Background Position background-position: defines the relative position of the background within the element
Background Size background-size: defines the size of the background
Background Repeat background-repeat: defines how the background image will be repeated horizontally and/or vertically
Background Origin background-origin: defines the position of the background in terms of the element structure eg. border, padding, content box
Background Clip background-position: defines the outermost bounds of the background in terms of element structure
Background Attachment background-attachment: defines how the background behaves with scrolling
Background background: shorthand property abbreviated method of styling a property by defining multiple values at once with values defined in the order following the order of the properties listed above (color, image, position, etc.)

Background Image

Background Color Values
Property Value Description
background-image: url(x) image value where x is the image location path
background-image: linear-gradient(x) linear gradient value where x is in the syntax linear-gradient(direction/angle, color 1, color 2, etc.)
background-image: radial-gradient(x) radial gradient value where x is in the syntax radial-gradient(direction/angle, color 1, color 2, etc.)
