What is flexbox?

Flexbox is short for Flexible Box Container Layout. It is a webpage layouting method that makes use of flex containers and flex items within those containers.

Syntax: 
<div style="background-color: white; padding: 5px;">
  Flex Container
  <div style="display: flex; background-color: white;">
    <div style="background-color: rgb(75, 175, 228); margin: 10px; padding: 2px">Flex Item 1</div>
    <div style="background-color: rgb(75, 175, 228);; margin: 10px; padding: 2px">Flex Item 2</div>
    <div style="background-color: rgb(75, 175, 228);; margin: 10px; padding: 2px">Flex Item 3</div>
  </div>
</div>

Flex Container

The flex container is the parent element containing flex items. Adjusting its properties will affect how the flex items are arranged within.

Properties

Flexbox Container Properties
Property Syntax Description
Display display: sets how the container behaves with surrounding elements: <flex> for block or <inline-flex> for inline
Flex Direction flex-direction: sets the direction of the flex items
Flex Wrap flex-wrap: determines whether flex items wrap go to the next line when there is insufficient space
Flex Flow flex-flow: shorthand property for flex direction and flex wrap
Justify Content justify-content: sets horizontal alignment (along main axis) of flex items
Align Items align-items: sets vertical alignment (along cross axis) of flex items
Align Content align-content: sets alignment if flex items wrap

Examples

Syntax: 

  color: white;
  height: 300px;
  text-align: center;
  display: flex;
  flex-direction: row;
  justify-content: flex-end;
  flex-wrap: wrap-reverse;
  align-items: stretch;
  align-content: stretch
