What are multimedia elements?

Multimedia elements include content other than text, such as: images, videos, and audio.

All three media can be inserted using their respective start tags, but only images do not have an end tag, making the image element a void element.

Attributes

The source attribute is the one common among all three media elements. It is applied using src followed by the local file path e.g. C:\Users\Name\Pictures\File.png or URL.

Images and videos must be inserted with the width and height attributes specified. These define their pixel dimensions. If only one of these is defined, the other dimension will adjust accordingly to maintain the original aspect ratio.

Lastly, videos and audio may be inserted along with the controls, autoplay, and muted attributes. controls display play, pause, and volume buttons; autoplay instructs the browser to play the media immediately upon the running of the webpage; and muted mutes the media by default.

While there are many file types for each media, only selected ones are compatible in browsers (see table).

Medium Tags Attributes File Types
Image <img> src width, height alt APNG, Animated Portable Network Graphics GIF, Graphics Interchange Format ICO, Microsoft Icon JPEG, Joint Photographic Expert Group PNG, Portable Network Graphics SVG, Scalable Vector Graphics
Video <video></video> src, width, height, controls, loop, autoplay, muted MPEG-4/MP4, Moving Pictures Expert Group OGG, WebM, Web Media
Audio <audio></audio> src, controls, loop, autoplay, muted MP3, OGG, WAV, Waveform Audio File Format

Inline Frame

The inline frame or iframe element allows webpages and files to be embedded into a nested browsing context smaller window inside the current window (viewport) within the webpage.

This nested browsing context enables interaction with the content inside the iframe that does not affect the larger window surrounding it.

Tags Attributes File Types
<iframe></iframe> src, width, height Image APNG, GIF, GIF, ICO, JPEG, PNG, SVG
Video MP4, OGG, WebM,
Audio MP3, OGG, WAV
Document DOCX, PPTX, XLSX, TXT, PDF
Web Content APNG, GIF, GIF, ICO, JPEG, PNG, SVG
